CSB541 Lab08 - Internet Addresses

Lab 08

  1. Download and unzip lab08.zip

    Remind: set paths and check versions (see mypath.bat)

  2. Compile Time.java, DictClient.java, LowPortScanner2.java, and SocketInfo.java 
    > javac *.java

  3. Run Time.java (Example 8-3), DictClient.java (Example 8-4), LowPortScanner2.java (Example 8-5) and SocketInfo.java (Example 8-6) by lab08run.bat 
    > java Time
> java DictClient computer network programming
> java LowPortScanner2
> java SocketInfo www.nuk.edu.tw www.google.com
    Check the results and make sure you understand what and how they do.

  4. Develop a new class A1035500Lab08 that extends DictClient.java (Example 8-4) to read inputs from the keyboard rather than from the command line.
    Finish the program by input 'quit'.

    The execution results look like:
    >javac A1035500Lab08.java
>java A1035500Lab08
gold	<------ your input
gold [gould]
	:
	:

lead	<------ your input
lead [led]
	:
	:

quit	<------ your input

    Upload the output screenshot of the compiling and execution steps.
