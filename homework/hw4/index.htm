CSB541 Lab04 - Internet Addresses

Lab 04

  1. Download and unzip lab04.zip

    Remind: set paths and check versions (see mypath.bat)

  2. Compile AllByName.java, IPCharacteristics.java, InterfaceLister.java, LookupTask.java, PooledWeblog.java, and SpamCheck.java 
    > javac *.java

  3. Run InterfaceLister.java (Example 4-8), IPCharacteristics.java (Example 4-6), AllByName.java, SpamCheck.java (Example 4-9), and PooledWeblog.java (Example 4-11) by lab04run.bat 
    > java InterfaceLister
> java IPCharacteristics www.csie.nuk.edu.tw
> java AllByName www.csie.nuk.edu.tw www.youtube.com
> java SpamCheck www.nuk.edu.tw www.google.com
> java PooledWeblog lab04log.txt
    Check the results and make sure you understand what and how they do.

  4. Develop a new class A1035500Lab04 that extends SpamCheck.java (Example 4-9) to use getAllByName() instead of getByName().
    You may refer to AllByName.java.
    The execution results look like:
    >javac A1035500Lab04.java
>java A1035500Lab04 www.csie.nuk.edu.tw www.youtube.com

www.youtube.com/64.233.189.190 appears legitimate.
www.youtube.com/64.233.189.91 appears legitimate.
